# 一、项目概述
《明日基石》（英文名 Foundation of Tomorrow）是一款面向高中生的教育类知识答题游戏。游戏以时代穿越为叙事主线，将上海高考等级考/合格考知识点嵌入五类职业角色的答题闯关中，让玩家在游戏中巩固学科知识，体验从 1950 年代到 2080 年代的文明演进。
游戏名称"明日基石"寓意：今天学习的每一道题、积累的每一个知识点，都是构筑未来的基石。五种职业对应不同的学科组合，玩家可以选择自己感兴趣的领域展开答题之旅。
# 二、设计理念
## 2.1 核心目标
- 轻量化学习：利用碎片时间答题巩固知识点，每局约 15～20 分钟
- 游戏化激励：通过打分、升级、称号、时代推进等机制维持学习动力
- 沉浸式体验：每个时代适配不同字体、视觉风格和背景情景，营造穿越感
- 自主掌控：支持手动存档、错题回顾、导出打印，学生可自主安排复习节奏
## 2.2 设计原则
知识点对标上海高考考纲与等级考要求，题目由团队成员根据沪教版教材、历年真题和考纲编写。所有游戏数据仅存储在用户本地浏览器中，不收集任何个人信息，可离线使用。
界面采用暗色主题，配色以金色（#ffd200）为主色调，配合大地色系和时代主题色，在降低视觉疲劳的同时营造"历史沉淀感"。
# 三、游戏架构
游戏采用纯前端架构，HTML + CSS + JavaScript 实现，所有数据存储在浏览器的 localStorage 中，无需服务器和网络连接。
核心文件结构：
- index.html — 主页面，包含 UI 框架、存档系统、错题本、登录系统、免责声明等所有功能模块
- game-logic2.js — 游戏引擎与题库，包含五个职业各三个时代的全部题目与游戏逻辑
游戏流程：用户打开页面 → 选择学段（高一/高二/高三）→ 选择职业 → 进入答题。每个职业经历三个时代（回望历史→当代风云→赛博未来），每个时代 5 道题，共 15 题，满分 150 分。答题结束后显示结算画面和最终等级。
# 四、职业系统
游戏提供五种职业，每种职业对应两门学科的组合：
- 🌾 农民（化学 + 政治）——适合对农学、化学和时政感兴趣的学生
- 📰 记者（语文 + 英语）——适合语言表达能力强、关注社会热点的学生
- 💉 医生（生物 + 地理）——适合对生命科学和自然环境感兴趣的学生
- 💻 程序员（信息科技 + 历史）——适合对计算机技术和历史变迁感兴趣的学生
- 🔧 工程师（数学 + 物理）——适合擅长逻辑推理和物理建模的学生
开始游戏前可选择学段（高一/高二/高三），系统根据选择的学段筛选对应难度的题目。高一只显示高一考点，高二包含高一+高二，高三包含全部三个年级的题目。
# 五、答题机制与评分
## 5.1 题型说明
- 单选题：四个选项中选一个正确答案，选对得 10 分
- 不定项选择题：1～2 个正确答案，全部选对得 10 分，漏选得 5 分，多选/错选不得分
- 多选题：2～3 个正确答案，需全部选对得 10 分
- 填空题：手动输入答案，正确得 5 分（自动判题，忽略大小写和空格差异）
## 5.2 等级与总分
每 50 分升一级，等级无上限。答题过程中进度条显示当前等级内的进度（0～50 分）。升级时弹出庆祝动画：金色旋转边框、撒纸屑特效。
单局满分 150 分（3 时代 × 5 题 × 10 分）。结算时根据得分率分为三档：≥70% 获得"大师"评价，≥40% 获得"合格"评价，<40% 获得"初出茅庐"评价。
# 六、时代推进系统
游戏的叙事核心是"时代穿越"——玩家依次经历三个历史时代：
- 回望历史——📜 回望历史（1950s-1970s）：对应新中国建设初期，使用 Noto Serif SC（宋体感），营造复古年代氛围。
- 当代风云——🌐 当代风云（1990s-2050s）：对应改革开放深化时期，使用 DotGothic16，呈现现代简洁风格。
- 赛博未来——🤖 赛博未来（2080s）：对应未来科幻时代，使用 Share Tech Mono（等宽科技感），配合青色高亮。
游戏区域顶部有时代时间轴组件，直观显示当前所在时代、已完成时代和未解锁时代。已完成时代显示绿色对勾，当前时代显示金色高亮，未解锁时代显示灰色锁定状态。每个时代开始前有情景叙述，将玩家代入时代背景中。
每道题目上方都有一个基于关键词自动匹配的情景句。系统内置 60+ 关键词-情景映射，覆盖化学、政治、语文、英语、生物、地理、信息科技、历史、数学、物理等全部学科。
# 七、称号系统
累计星星每达到 300 星解锁一个新称号，称号显示在开始页面。称号采用四字成语或短词 + "明日基石"的形式，如"勤学不辍的明日基石"、"探赜索隐的明日基石"等。
300 星以下不显示称号。称号文字使用彩虹色三色渐变，从暖色过渡到冷色再到暖色，形成视觉循环。
目前共有 16 个称号等级，最高至 4800 星，覆盖从"勤学不辍"到"光照未来"的完整成长路径。
# 八、错题本系统
所有答错或未完全答对的题目（包括部分正确）自动记录到错题本中。记录包含题目文本、选项、用户所选答案、正确答案、得分和知识点解析。
错题面板按题目文本去重后分为"未掌握"和"已掌握"两类。未掌握：从未答对过的题目，以正常亮度显示；已掌握：曾答错但之后答对过的题目，以半透明显示。通过时间戳比较判断。
**🔁 单题重做**：单击任意错题（未掌握或已掌握均可），直接进入该题的独立重做模式。重做时显示正确答案和知识点解析（金色标注），累加总星数但**不影响当前游戏等级**。部分正确显示黄色，完全正确显示绿色，错误显示红色。答完点击「返回错题本」回到面板。
错题面板底部提供两种导出方式：导出无解析版（仅含题目和选项，适合自测）和导出有解析版（包含题目、选项、正确答案和知识点解析，适合复习）。导出文件为 HTML 格式，自动打开打印窗口或触发下载。
# 九、存档与账户系统
所有游戏数据存储在浏览器的 localStorage 中，使用两个存储键：foundation_of_tomorrow_save（游戏存档）和 lifeSimulator_records（答题记录）。
存档功能包括：结算时自动保存最高分、游戏过程中静默保存当前进度、手动保存、退出时弹窗提醒。支持多账号切换（适合多人共用设备），账号仅用于标识用户名，游戏数据共享。首次访问不自动创建账号，用户可通过"切换账号"按钮创建或选择账号。
开始页面底部有"清空历史记录"按钮，一键清除所有存档、答题记录和账号信息。
# 十、UI/UX 设计
主色调为金色（#ffd200），搭配深色背景（#0a0a1a）。五种职业卡片各有渐变色：农民绿色系、记者蓝色系、医生粉色系、程序员青色系、工程师橙色系。
游戏使用 7 种 Google Fonts 字体：Orbitron（标题LOGO）、Press Start 2P（英文辅助）、DotGothic16（主UI）、ZCOOL QingKe HuangYou（情景对话）、Noto Serif SC（复古时代）、Share Tech Mono（赛博数据）、PixelMplus12（年代标识）。
页面底部固定制作团队水印，右上角固定"声明 & 玩法"按钮。升级时弹出全屏遮罩动画，使用 CSS keyframes 实现，无额外库依赖。
# 十一、开发与迭代过程
游戏最初为单页 HTML 应用，含约 30 题。经多次迭代逐步扩充至 200+ 题，覆盖五个职业全部三时代，增加不定项选择和多选题，按上海教材年级标注难度。
关键迭代包括：选项优化（删除解释性后缀和括号内容）、时代字体（三时代三种字体方案）、称号系统（16 级三色彩虹渐变）、存档系统（手动/自动/静默三种方式）、错题本（去重分类+单题重做+导出打印）、免责声明弹窗和账户系统。
测试修复了多个问题：选项补长方案改为题库层精简、打印弹窗被拦截改用 Blob URL、旧用户总星数为 0 通过迁移函数修复、重做模式返回弹窗改为静默保存+视图恢复等。
# 十二、技术栈与依赖
- HTML5 + CSS3 + JavaScript (ES5，兼容主流浏览器)
- localStorage 持久化存储
- Google Fonts（7 种，SIL OFL / Apache 2.0 开源协议，可免费商用）
- 无第三方框架/库依赖，零外部请求（字体 CDN 除外）
游戏可离线使用（首次加载字体后）。在 Chrome、Edge、Firefox 最新版及微信内置浏览器中测试通过。
# 十三、展示要点（可直接念的总结）
## 开场（30 秒）
"大家好，今天我们带来的作品是《明日基石》——一款知识答题驱动的时代穿越游戏。玩家可以从五种职业中选择一种，在三个历史时代中答题闯关，每道题都对应上海高考的考点。"
## 核心演示（2 分钟）
"首先看到的是职业选择页面。这里有五个职业卡片，每个对应两个学科。我们选择农民这个职业进入答题——可以看到左上角的时代标签，游戏区域顶部的时间轴显示我们正处于第一个时代回望历史。答题后会有得分反馈和知识点解析，升级时弹出庆祝动画。"
## 错题本演示（1 分钟）
"回到开始页面，点击下方的▼展开错题面板。这里按未掌握和已掌握分类展示所有错题，点击任意错题可以直接进入重做模式。底部还有导出打印功能，可以选择有解析版或无解析版，方便课后复习。"
## 亮点总结（30 秒）
"我们的游戏有几个亮点：第一，沉浸式的时代字体切换，三个时代各有不同视觉风格；第二，累计星星解锁称号，用彩虹色渐变带来持续激励；第三，完整的错题管理系统，支持单题重做和导出打印。所有数据都保存在本地，不需要联网，可以随时随地进行练习。"
